# Database-Setup-and-Schema-Design
Basic SQL database for a library system containing tables for members, books, records, and staff. Supports storing and managing core library information efficiently.

## Features 
- Member information storage
- Book catalog with details
- Borrow and return records
- Staff information management
- Foreign key relationships

## Technologies Used
- MySQL
- MySQL Workbench

## Project Structure
- members - Stores member details.
- books - Stores book details.
- records - Stores borrow and return history.
- staffs - Stores staff information.

## Steps to run
- Open MySQL Workbench
- Create a new database named "library_db"
- Run the SQL table creation script - members, books, records and staffs
- Verify tables using "SHOW TABLES"
